# 02 — About the Cohort Section

## Purpose
Answer "what exactly is this?" and "why is it free?" before the user scrolls away.
Build trust, create urgency, establish community identity.

---

## Section ID
`#about`

## Layout
Three-column card grid on desktop. Single column stacked on mobile.

---

## Section Header

```
EYEBROW: ABOUT THE COHORT
HEADLINE: Not a course. A community.
SUBHEADLINE: The first cohort of what will become Abu Dhabi's
             premier Data Engineering community — and you can be part of it from day one.
```

Arabic:
```
EYEBROW: عن الكوهورت
HEADLINE: مش كورس. مجتمع.
SUBHEADLINE: الكوهورت الأول لأكبر مجتمع Data Engineering في أبوظبي —
             وانت ممكن تكون جزء منه من أول يوم.
```
